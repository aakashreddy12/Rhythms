EVENT COVER IMAGES FOLDER
=========================

Place event cover photos here with these exact file names:

✅ REQUIRED FILE NAMES:
- cultural-fest-2024.jpg
- inter-college-comp.jpg
- rhythms-showcase.jpg
- dance-workshop.jpg
- music-night.jpg
- theatre-festival.jpg

📏 IMAGE REQUIREMENTS:
- Size: 800x600px minimum
- Format: JPG, PNG, or WebP
- Aspect ratio: 4:3 (landscape)
- File size: Under 3MB each

🔧 HOW TO ADD:
1. Copy your event cover photos to this folder
2. Rename them to match the exact names above
3. Refresh the website - covers appear on Gallery page!

📸 WHAT ARE COVER IMAGES:
These are the main photos that represent each event on the Gallery page.
Choose the best/most representative photo from each event.

🎯 TIPS:
- Use high-quality, well-lit photos
- Choose photos that represent the event well
- Avoid blurry or dark images
- Landscape orientation works best

Delete this README.txt file after adding your photos.
