EVENT PHOTO COLLECTIONS FOLDER
==============================

Create folders for each event and add photos inside them.

✅ REQUIRED FOLDER STRUCTURE:

cultural-fest-2024/
├── photo-1.jpg
├── photo-2.jpg
├── photo-3.jpg
└── ... (up to photo-12.jpg)

inter-college-comp/
├── photo-1.jpg
├── photo-2.jpg
└── ... (up to photo-8.jpg)

rhythms-showcase/
├── photo-1.jpg
├── photo-2.jpg
└── ... (up to photo-15.jpg)

dance-workshop/
├── photo-1.jpg
└── ... (up to photo-6.jpg)

music-night/
├── photo-1.jpg
└── ... (up to photo-10.jpg)

theatre-festival/
├── photo-1.jpg
└── ... (up to photo-9.jpg)

📏 IMAGE REQUIREMENTS:
- Size: 600x600px minimum
- Format: JPG, PNG, or WebP
- Any aspect ratio (square preferred)
- File size: Under 2MB each

�� HOW TO ADD:
1. Create event folder (e.g., "cultural-fest-2024")
2. Copy your event photos into the folder
3. Rename them as photo-1.jpg, photo-2.jpg, etc.
4. Refresh website - photos appear in Gallery!

📸 NAMING CONVENTION:
- Always start with photo-1.jpg
- Number sequentially: photo-1.jpg, photo-2.jpg, photo-3.jpg
- Use lowercase
- No spaces or special characters

🎯 TIPS:
- Add photos in order of importance/quality
- You can add more photos than the default numbers
- Choose diverse photos showing different aspects of the event
- Include both wide shots and close-ups

🆕 TO ADD NEW EVENT:
1. Create new folder with event name (lowercase, hyphens)
2. Add photos as photo-1.jpg, photo-2.jpg, etc.
3. Update the code in client/lib/imageManager.ts

Delete this README.txt file after adding your event folders.
