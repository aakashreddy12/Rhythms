LOGO FILES FOLDER
=================

Place your Rhythms logo here with this exact file name:

✅ REQUIRED FILE NAME:
- rhythms-logo.png

📏 LOGO REQUIREMENTS:
- Size: 800x800px minimum
- Format: PNG (preferred for transparency)
- Background: Transparent preferred
- File size: Under 1MB

🔧 HOW TO ADD:
1. Copy your logo file to this folder
2. Rename it to "rhythms-logo.png"
3. Refresh website - logo appears in navbar and hero background!

🎯 WHERE LOGO APPEARS:
- Navbar (top-left, small version)
- Hero section background (large, blurred version)

💡 TIPS:
- PNG format preserves transparency
- High resolution ensures quality at all sizes
- Square aspect ratio works best
- Logo should work on both light and dark backgrounds

🎨 DESIGN NOTES:
- Logo appears small (40x40px) in navbar
- Logo appears large and blurred in hero background
- Make sure logo is legible at small sizes

Delete this README.txt file after adding your logo.
